# Windows Password Policy & Account Security Lab

This repository documents a hands-on lab where I configured and tested password policies in both **local Windows systems and domain environments**.

The objective of this lab was to gain practical experience with **Windows administration tools used to enforce authentication security and manage user accounts**.

## Skills Practiced

- Configuring Local Password Policy
- Setting password complexity requirements
- Configuring password history and expiration
- Implementing account lockout policies
- Testing password policy enforcement
- Unlocking locked-out local user accounts
- Configuring domain password policies using Group Policy
- Unlocking locked-out domain user accounts

## Technologies Used

- Windows Server
- Active Directory
- Group Policy Management
- Local Security Policy
- Windows Administrative Tools
